WxPenguin
================

WeChat client for linux, based on Web WeChat.
All rights of wechat reserved by Tencent.

Modules
--------------

+ WebWx -- Web wechat API in C++
+ WxQt -- Adapter for Qt
+ UI -- Written with Qt
+ Plugin -- Enable plugin development, such as robot
+ Database -- Plan to use sqlite
